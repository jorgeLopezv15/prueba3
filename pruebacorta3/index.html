JAVA

Java es un lenguaje de programación de alto nivel, orientado a objetos, y concurrente, diseñado para ser lo más independiente posible de la plataforma hardware en la que se vaya a ejecutar, siguiendo el principio de "escribe una vez, ejecuta en cualquier lugar" (WORA, por sus siglas en inglés). Fue desarrollado por Sun Microsystems (que posteriormente fue adquirida por Oracle Corporation) y lanzado por primera vez en 1995.

html

HTML, que significa HyperText Markup Language (Lenguaje de Marcado de Hipertexto), es el lenguaje estándar utilizado para crear y diseñar páginas web. Es la base fundamental sobre la cual se construyen la mayoría de los sitios web en Internet. HTML no es un lenguaje de programación, sino un lenguaje de marcado que define la estructura y el contenido de una página web mediante el uso de etiquetas (tags).

Css

CSS, que significa Cascading Style Sheets (Hojas de Estilo en Cascada), es un lenguaje utilizado para definir la presentación y el diseño de documentos HTML. Mientras que HTML se encarga de la estructura y el contenido de una página web, CSS se utiliza para controlar su apariencia, incluyendo aspectos como colores, fuentes, márgenes, tamaños, posicionamiento y más.

framework

un framework es un conjunto de herramientas, bibliotecas, convenciones y buenas prácticas que proporcionan una estructura base para desarrollar aplicaciones o sistemas de software de manera más eficiente y organizada. Los frameworks están diseñados para simplificar el proceso de desarrollo, reducir la redundancia de código y permitir a los desarrolladores centrarse en la lógica específica de su aplicación en lugar de preocuparse por aspectos repetitivos o de bajo nivel.
